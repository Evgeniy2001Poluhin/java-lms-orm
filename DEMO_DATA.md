## Примеры использования

### Запись студента на курс

```bash
curl -X POST "http://localhost:8080/api/enrollments?studentId=2&courseId=1"
```

### Отправка решения задания

```bash
curl -X POST "http://localhost:8080/api/assignments/1/submit?studentId=2" \
  -H "Content-Type: application/json" \
  -d '{"content": "Мое решение задания"}'
```

### Прохождение теста

```bash
curl -X POST "http://localhost:8080/api/quizzes/1/take?studentId=2&timeTaken=15" \
  -H "Content-Type: application/json" \
  -d '{
    "1": [1],
    "2": [3]
  }'
```

Где:
- `"1": [1]` - на вопрос с ID=1 выбран ответ с ID=1
- `"2": [3]` - на вопрос с ID=2 выбран ответ с ID=3

## Расширение демо-данных

Вы можете добавить свои данные, отредактировав файл:
`src/main/java/ru.polukhin/learningplatform/config/DataInitializer.java`

Или использовать REST API для создания новых данных после запуска приложения.

## Сброс данных

Данные автоматически пересоздаются при каждом запуске приложения с профилем `dev` 
(используется `spring.jpa.hibernate.ddl-auto=create-drop`).

Для сохранения данных между запусками измените настройку на:
```properties
spring.jpa.hibernate.ddl-auto=update
```

## Проверка данных

После запуска приложения вы можете проверить созданные данные:

```bash
# Все пользователи
curl http://localhost:8080/api/users

# Все курсы
curl http://localhost:8080/api/courses

# Курс с модулями
curl http://localhost:8080/api/courses/1/with-modules

# Тест с вопросами
curl http://localhost:8080/api/quizzes/1/with-questions
```

Или через Swagger UI: http://localhost:8080/swagger-ui.html
